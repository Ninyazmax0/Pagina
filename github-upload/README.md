# Farmacia Santa Cruz - Sistema de Inventario

Sistema de gestión de inventario para la Farmacia Santa Cruz, desarrollado como parte del proyecto académico de la institución **The Repair of System**.

## Características

- Gestión de productos (medicamentos)
- Control de inventario y stock
- Registro de compras a proveedores
- Registro de ventas
- Alertas de caducidad
- Reportes de inventario
- Control de acceso por usuarios

## Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Instalación

```bash
npm install
npm run dev
```
